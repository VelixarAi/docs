# What is Velixar?

Velixar is **memory infrastructure for AI systems**. It gives your LLM-powered applications persistent, encrypted memory with a knowledge graph, immutable audit trails, and automatic consolidation — so your AI remembers everything, connects the dots, and gets smarter over time.

## Why Velixar?

LLMs are stateless. Every conversation starts from zero. RAG retrieves documents — but documents aren't memory. Velixar gives AI systems actual memory: persistent, structured, encrypted, and auditable.

| Without Velixar | With Velixar |
|---|---|
| User repeats preferences every session | Preferences recalled automatically |
| No context between conversations | Full knowledge graph across sessions |
| Generic responses for every user | Personalized responses from memory |
| Manual RAG pipeline management | Semantic search + graph traversal built in |
| No audit trail for AI decisions | Every recall logged with cryptographic proof |
| Memory data stored in plaintext | Envelope-encrypted with workspace-scoped keys |

## How It Works

<Steps>
  <Step title="Store">
    Send memories to the API — facts, preferences, context, decisions. Each memory is encrypted, embedded, and indexed. Entities are extracted into a knowledge graph automatically.
  </Step>
  <Step title="Recall">
    Query with natural language. Velixar returns the most relevant memories, enriched by knowledge graph traversal. Recalled memories automatically increase in salience.
  </Step>
  <Step title="Connect">
    The knowledge graph links related concepts across all your memories. Recall "user's tech stack" and get the database, framework, hosting, and deployment details — all connected.
  </Step>
  <Step title="Consolidate">
    Velixar periodically synthesizes related memories into higher-order knowledge — turning hundreds of individual facts into structured, retrievable summaries.
  </Step>
  <Step title="Audit">
    Every operation is logged in a tamper-evident, hash-chained audit trail. Know exactly what your AI knew, when it knew it, and how that knowledge changed.
  </Step>
</Steps>

## Key Features

- **Encrypted by default** — Envelope encryption with workspace-scoped keys. Your embeddings are unreadable, even to us.
- **Knowledge graph** — Memories aren't isolated vectors. Velixar builds a directed graph of relationships between entities and concepts.
- **Immutable audit trail** — Hash-chained logging of every store, recall, update, and promote operation.
- **Deep consolidation** — Automatic synthesis of related memories into higher-order knowledge.
- **Infinite memory** — Nothing is ever deleted. Salience scoring surfaces active knowledge while keeping everything accessible.
- **Tiered storage** — Pin critical facts (tier 0), session context (tier 1), semantic (tier 2), org-wide (tier 3).
- **Auto-extraction** — Automatically extract memories from conversations with deduplication and conflict resolution.
- **Multi-tenant** — Complete workspace isolation with separate encryption keys, graph namespaces, and audit trails.
- **SDKs** — Python and JavaScript clients, CLI, MCP server, plus LangChain integration.
