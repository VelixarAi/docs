---
title: "Search Graph"
api: "POST /graph/search"
description: "Search entities by name or type."
---

Find entities in the knowledge graph by name, type, or both.

## Request Body

<ParamField body="query" type="string">
  Search query — matches entity names and descriptions.
</ParamField>

<ParamField body="entity_type" type="string">
  Filter by entity type (e.g. `person`, `technology`, `project`).
</ParamField>

<ParamField body="limit" type="integer" default="20">
  Maximum results.
</ParamField>

## Response

```json
{
  "entities": [
    {
      "id": "node-abc",
      "entity_type": "person",
      "label": "Alice",
      "properties": { "role": "Tech Lead" },
      "relevance": 0.92
    }
  ]
}
```
