---
title: "Delete Memory"
api: "DELETE /v1/memory/{id}"
description: "Delete a memory. Requires memory:delete scope."
---

## Path parameters

<ParamField path="id" type="string" required>
  The memory ID to delete.
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE https://api.velixar.ai/v1/memory/mem_abc123 \
  -H "Authorization: Bearer vlx_your_key"
```

```python Python
resp = requests.delete("https://api.velixar.ai/v1/memory/mem_abc123",
    headers={"Authorization": "Bearer vlx_your_key"})
```

```javascript JavaScript
const resp = await fetch("https://api.velixar.ai/v1/memory/mem_abc123", {
  method: "DELETE",
  headers: { "Authorization": "Bearer vlx_your_key" }
});
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": "mem_abc123",
  "deleted": true
}
```

```json 404
{
  "error": "not_found",
  "message": "Memory not found"
}
```
</ResponseExample>
