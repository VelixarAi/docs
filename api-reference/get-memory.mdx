---
title: "Get Memory"
api: "GET /v1/memory/{id}"
description: "Retrieve a specific memory by ID."
---

## Path parameters

<ParamField path="id" type="string" required>
  The memory ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.velixarai.com/v1/memory/mem_abc123 \
  -H "Authorization: Bearer vlx_your_key"
```

```python Python
resp = requests.get("https://api.velixarai.com/v1/memory/mem_abc123",
    headers={"Authorization": "Bearer vlx_your_key"})
```

```javascript JavaScript
const resp = await fetch("https://api.velixarai.com/v1/memory/mem_abc123", {
  headers: { "Authorization": "Bearer vlx_your_key" }
});
```
</RequestExample>
