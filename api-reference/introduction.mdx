## Base URL

```
https://api.velixarai.com
```

## Authentication

All endpoints require a Bearer token:

```bash
Authorization: Bearer vlx_your_api_key
```

Get your key from [Settings → API Keys](https://velixarai.com/settings).

## OpenAPI Spec

A full [OpenAPI 3.1 specification](https://github.com/VelixarAi/velixar-mcp-server/blob/main/openapi.json) is available for code generation and API exploration.

## Response Format

All successful responses return JSON:

```json
{
  "id": "abc-123",
  "content": "...",
  "tags": ["decision", "architecture"]
}
```

## Error Format

```json
{
  "error": "Human-readable description",
  "status": 400
}
```

| Code | Meaning |
|------|---------|
| 400 | Invalid parameters |
| 401 | Missing or invalid API key |
| 403 | Key lacks required scope |
| 404 | Resource not found |
| 429 | Rate limit exceeded (check `Retry-After` header) |
| 500 | Server error |
