We're excited to announce that Velixar Memory is now published on the [official MCP Registry](https://github.com/mcp) as `io.github.VelixarAi/memory`.

## What This Means

Any MCP-compatible client can now discover and install Velixar directly from the registry. No manual configuration needed — just search for "velixar" or "memory" in your client's MCP server browser.

## Supported Clients

- Claude Desktop
- Kiro
- Cursor
- Windsurf
- Continue
- Any client that supports the MCP Registry

## Quick Install

If you prefer manual setup:

```bash
npm install -g velixar-mcp-server
```

Then add to your client config with your `VELIXAR_API_KEY` environment variable. [Full setup guide →](/blog/mcp-memory-setup-guide)
